# 页面交互优化及修复计划

## 1. 当前状态分析
- **足球光标特效**：现有的 SVG 光标未能良好呈现足球特征；当前有基于滚动的下坠特效；光标离开页面时依然停留在边缘；拖动时的扬尘效果不明显；启用了系统“减少动态效果”检测导致部分用户无法看到特效。
- **UI 样式**：登录页（及其他使用了基础组件的地方）的表单卡片、输入框、按钮均为直角或尖锐边缘，缺乏“明显的圆角”。
- **502 登录错误**：经排查，由于后端服务（端口 3001）未启动，导致前端 Vite 代理转发到 `/api/v1/auth/login` 时连接被拒绝（Bad Gateway）。

## 2. 计划修改内容

### 任务一：足球光标及动效优化 (SoccerFX.vue & svg)
1. **重绘足球 SVG**：更新 `SoccerFX.vue` 中的 `soccerSvg`，采用更逼真的蓝白相间五边形/六边形绘制逻辑，使其符合用户提供的参考图片。
2. **移除滚动下坠，增加点击发射**：
   - 移除 `triggerFall` 函数以及相关的 `scroll` 监听。
   - 新增 `shootBall(x, y)` 函数：当用户在页面空白处（非交互元素）点击时，从点击位置向随机方向发射一个足球，触碰屏幕边缘反弹，反弹次数达到 3 次后淡出并移除。
3. **离开页面隐藏光标**：
   - 监听 `document` 的 `mouseleave` 和 `mouseenter` 事件。
   - 离开时将 `cursorEl` 的透明度设为 `0`，进入时恢复为 `1`。
4. **增强扬尘效果**：
   - 在 `spawnDust` 中增加生成的粒子数量，放大粒子初始缩放比例（`--s`），提高不透明度（`--o`），并降低触发扬尘的速度阈值，使得拖拽时扬尘更加明显。
5. **停用“减少动态效果”检测**：
   - 在 `onMounted` 的 `recompute` 方法中，移除 `!reduceMotion.matches` 的限制，只要支持精细指针（`finePointer.matches`）即可启用特效。

### 任务二：表单框、输入框、按钮圆角化
1. **修改 LoginPage.vue**：
   - 为 `.login-card` 增加 `border-radius: 24px`。
   - 为 `.form-input` 增加 `border-radius: 12px`，并添加全包围的边框 `border: 1px solid var(--border-strong)`。
   - 为 `.editorial-btn` 增加 `border-radius: 12px`。
2. **修改 Button.vue**：
   - 将 `.base-button` 的 `border-radius` 从 `0` 修改为 `12px`，统一全局基础按钮风格。

### 任务三：修复 502 登录错误
1. **启动后端服务**：
   - 使用终端命令在 `backend` 目录下执行 `npm run dev` 启动后端服务（端口 3001），解决代理无法访问的问题。

## 3. 验证步骤
- 刷新页面并观察光标是否变为新的足球图标。
- 将鼠标移出浏览器窗口，检查光标是否消失。
- 在页面空白处点击，检查是否会发射足球并在反弹 3 次后消失。
- 拖拽光标，观察扬尘是否显著增强。
- 检查登录页的表单卡片、输入框和按钮是否呈现明显的圆角。
- 尝试登录，确认不再报错 502 并能正常进行请求。
