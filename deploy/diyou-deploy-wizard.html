🏆 棣友档案站 · 极空间部署向导

跟着提示一步一步填，生成部署文件后复制到极空间即可

1网站访问方式

例如：diyou.yourdomain.com（不需要写 https://）
请填写正确的域名
打开极空间手机 App → 设备管理 → 查看形如 192.168.x.x 的地址
请填写类似 192.168.x.x 的 IP 地址

2极空间上的存放位置

在极空间文件管理里新建一个文件夹，例如 /SATA存储/docker/diyou。
然后点击“查询路径”按钮复制完整路径粘贴到这里。
路径不能为空

3安全密钥（JWT_SECRET）

可以点击下方按钮自动生成，也可以自己写一串复杂字符。请妥善保存。
请先生成或填写 JWT_SECRET

4管理员账号

请填写正确的邮箱地址
至少 8 位，必须同时包含大写字母、小写字母和数字。
密码不符合要求

5邮箱验证码（SMTP）

用户点击“发送验证码”后，系统会用这个邮箱把验证码发给用户。

请填写发件邮箱
不是登录密码！请前往 QQ 邮箱 → 设置 → 账户 → 开启 SMTP 后获取 16 位授权码。
请填写授权码

6镜像访问方式

如果您把镜像设成了 Private，需要先在 GitHub 上把客户的 GitHub 账号加入镜像的“读取”权限。
在客户 GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) 中生成，只勾选 read:packages。

✅ 生成成功！

请点击下方按钮下载 diyou-zspace-deploy.zip，然后按步骤操作。

查看生成的 .env 文件（可核对）
查看生成的 docker-compose.yaml
